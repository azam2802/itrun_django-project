Некоторые проекты без адаптации под мобильные устройства.
В папке videos хранятся видео демонстрации сайтов.

Для запуска React проектов:
1. В IDE открыть терминал и выполнить npm install
2. Выполнить комманду npm start


Для запуска Django проекта:
1. В IDE открыть терминал и выполнить py -m venv venv
2. Активировать виртуальное окружение ./venv/Scripts/activate
3. Далее выполнить установку Django и pillow pip install django pillow
4. Выполнить команду py ./manage.py runserver 
5. Перейти по указанному в терминале IP
